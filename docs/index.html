Emacs Configuration

はじめに

  • ここは @minoruGH の Emacs設定ファイルの一部を解説しているページです。
  • https://github.com/minorugh/emacs.d/ から jekyll を使ってGithub pages にWebサイトを構築しています。
  • 本ドキュメントは、@takaxpさんが公開されている takaxp.github.io/ の記事を下敷きにした模倣版です。
  • 執筆用途に特化してカスタマイズしていますので、コンセプトやキーバイドなどは極めて邪道思想になっています。

動作確認環境

以下の環境で使用しています。が、動作を保証するものではありません。

  • Debian 11.4 86_64 GNU/Linux
  • 自分でビルドした Emacs 27.2.50
